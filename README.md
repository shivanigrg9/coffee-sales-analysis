# Coffee Shop Sales Analysis (2025)

This project analyzes the 2025 sales of a fictional coffee shop chain using Excel, SQL and Power BI.

## Skills Used
- Excel (Cleaning, Pivot Tables, Dashboard)
- SQL (Aggregations, GROUP BY, Trends)
- Power BI (Interactive dashboard)
- Data Cleaning & EDA

## Business Questions
- What is the total revenue?
- Which items sell the most?
- Which city performs best?
- Daily/Monthly sales trend?

## Insights
- Coffee contributes ~78% of total revenue.
- Latte and Cappuccino are top sellers.
- Delhi has highest average order value.
